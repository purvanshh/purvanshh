## 🏆 Featured Projects

### [PRGuard-AI](https://github.com/purvanshh/PRGuard-AI)
&gt; Multi-agent GitHub PR review system with parallel Celery workers (Style, Logic, Security agents), AST-based code parsing via tree-sitter, confidence arbitration with IoU-based span clustering, and NVIDIA NIM LLM backend.

`Python` `FastAPI` `Celery` `Redis` `tree-sitter` `ChromaDB` `Docker`

---

### [AuditLend (ALICe)](https://github.com/purvanshh/AuditLend-Intelligence-Core--ALICe-)
&gt; Production-grade auditable credit decision engine with deterministic failure simulation, idempotent intake, asynchronous processing, encrypted PII storage, immutable audit logs, calibrated decision confidence, and an opt-in ML scorer with fallback guardrails.

`Python` `FastAPI` `Celery` `PostgreSQL` `Redis` `XGBoost` `SHAP` `Docker` `Prometheus`

---

### [DRISE](https://github.com/purvanshh/DRISE-experiments)
&gt; Document Retrieval & Intelligence with Structured Extraction — Layout-aware multimodal document parsing (LayoutLMv3 + PaddleOCR) that transforms PDFs and images into validated, structured JSON with per-field confidence scores and cross-field consistency checks.

`Python` `FastAPI` `LayoutLMv3` `PaddleOCR` `PyTorch` `Docker`

---

### [GitHub-RAG](https://github.com/purvanshh/Github-Rag)
&gt; RAG pipeline over GitHub repositories — indexes codebases into a vector store and enables natural language querying over code, PRs, and documentation using tree-sitter parsing, dependency/call graphs, and cross-encoder reranking.

`Python` `FastAPI` `ChromaDB` `OpenAI` `tree-sitter` `NetworkX` `Streamlit`
